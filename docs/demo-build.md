# Demo-сборка для itch.io

Demo — отдельная HTML5-редакция игры без Yandex SDK, Steam API, рекламы и
внутриигровых покупок.

```sh
npm run dev:demo
npm run build:demo
```

Готовая сборка создаётся в `dist-demo/`. Для публикации на itch.io нужно
упаковать содержимое этой папки в ZIP так, чтобы `index.html` находился в корне
архива.

Ограничения редакции собраны в
`src/shared/config/gameEdition.ts`. Demo использует localStorage с отдельным
ключом прогресса и не изменяет сохранение local/desktop-версии.

В demo доступны вступление, обучение, первая сюжетная миссия, таверна,
лидерборд и первые три уровня быстрых игр. Основной arcade-режим и PvP
показывают предложение перейти на полную версию.

Ссылка на страницу Steam хранится в `src/shared/config/steam.ts`.
